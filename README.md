# 🤝 گروه‌یار — بات مدیریت گروه بله

## 🛠 وضعیت واقعی این تحویل (شفاف و صادقانه)
وقتی فایل‌ها رو بررسی کردم، بخش زیادی از خواسته‌هات (۹ موقعیت متن، سایز دقیق، انتخاب خط دور، ۱۰۰ کلمه‌ی بازی دار با ۲ راهنما، سیستم امنیتیِ درست‌شده، تشخیص «گروه‌یار») از قبل به‌درستی نوشته شده بود. ولی موقع بررسی دقیق، **۳ باگ واقعی و مهم پیدا کردم** که همین الان رفعشون کردم:

### 🐛 باگ ۱ (احتمالاً دلیل اصلی «گیف ساخته نمی‌شه»)
دکمه‌های «خط دور نوشته داشته باشه یا نه» و «رنگ خط دور» اصلاً به هیچ هندلری در `main.py` وصل نبودن! یعنی وقتی روشون می‌زدی، بات هیچ واکنشی نشون نمی‌داد و کل روند ساخت گیف همون‌جا متوقف می‌شد. الان `gifoutline:` و `gifoutlinecolor:` رو به `build_application()` اضافه کردم.

### 🐛 باگ ۲: دکمه‌ی اینلاین ورود به ساخت گیف
تابع `start_gif_maker_from_callback` نوشته شده بود ولی هیچ‌جا نه رجیستر شده بود، نه دکمه‌ای براش نشون داده می‌شد. الان:
- بعد از تأیید عضویت در پیوی، دکمه‌ی «🎬 ساخت گیف» نشون داده می‌شه
- توی پاسخ «راهنما»/«شروع» در پیوی هم همین دکمه هست

### 🐛 باگ ۳: مقاومت در برابر هنگ (سطح شبکه)
`concurrent_updates` و تایم‌اوت‌های صریح روی Application تنظیم نشده بودن (پیش‌فرض پی‌تی‌بی، آپدیت‌ها رو یکی‌یکی و سریال پردازش می‌کنه؛ یعنی اگه ساخت یه گیف طول بکشه، کل بات برای همه‌ی کاربرا معطل می‌مونه). الان `concurrent_updates(4)` + تایم‌اوت‌های connect/read/write/pool اضافه شد.

### ✅ چیزهایی که از قبل درست بودن (تأیید و تست شد)
- سیستم امنیتی: فقط تکرارِ **همون دکمه/همون دستور** مسدود می‌کنه (نه مجموع اکشن‌های متفاوت) — دقیقاً طبق توضیح اصلاحی‌ت
- «شروع» در پیوی: حداکثر ۵ بار در ۶۰ ثانیه، وگرنه بلاک
- تشخیص «گروه‌یار» چه با نیم‌فاصله چه با فاصله‌ی معمولی
- بازی «دار»: ۱۰۰ کلمه، هرکدوم دقیقاً ۲ راهنمای کامل (تست شد، صفر مورد ناقص)
- ساخت گیف: شبکه‌ی ۹تایی موقعیت، سایز دقیق (۱۰ تا ۲۰۰ پیکسل)، رنگ و روشن/خاموش بودن خط دور، پردازش در ترد جدا (ضد هنگ) با سقف ۲ کار هم‌زمان و timeout ۹۰ ثانیه
- `base_file_url` درست تنظیم شده بود (آدرس دانلود فایل از بله، نه تلگرام)

## ⚙️ نصب

```
pip install -r requirements.txt
```


## 🧪 تست قبل از اجرای واقعی
1. توی پیوی بات، «ساخت گیف» رو بزن، تا آخر مراحل برو، روی «✅ بله، خط دور داشته باشه» بزن — الان باید رنگ خط دور رو ازت بپرسه (قبلاً همین‌جا گیر می‌کرد).
2. یه عکس واقعی بفرست و ببین گیف واقعاً برات میاد.
3. یه اکانت دیگه رو ۴ بار پشت‌سرهم روی **همون دکمه‌ی خاص** بزن، ببین بار چهارم جواب نمیده؛ ولی اگه ۴ تا دکمه‌ی **متفاوت** بزنه نباید بلاک بشه.
4. توی پیوی، «شروع» رو ۶ بار پشت‌سرهم بزن، ببین بار ششم جواب نمیده.
5. «دار» رو شروع کن، وسط بازی بنویس «راهنمایی»، دوبار امتحان کن، سومی باید بگه راهنمایی تموم شده.
6. «گروه یار» (با فاصله‌ی معمولی) رو بنویس، ببین جواب میده.

هر ارور یا رفتار عجیبی دیدی، کل متن ترمینال یا اسکرین‌شات پیام بات رو بفرست تا دقیق بررسی کنم.
